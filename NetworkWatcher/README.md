# NetworkWatcher

NetworkWatcher is a small Windows application that monitors your network connections, primarily designed to inform user that his PC is connected to VPN. It sits in the system tray and alerts you whenever a new non-ignored network connection is detected.

## Features

- Monitors active network connections continuously.
- Sends notifications via the tray icon when any new non-ignored connection appears.
- Can run in the background and start automatically with Windows.
- Tray icon provides quick access to application controls.

## Installation

1. Download the latest release `.exe` file from the repository.
2. Run the application; it will appear in the system tray.
3. Use the tray icon context menu to enable or disable automatic startup.

## Usage

- Keep NetworkWatcher running in the background.
- Watch the tray icon for alerts about non-ignored connections.
