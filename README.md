# NetworkWatcher

NetworkWatcher is a small Windows application that monitors your network connections, primarily inform user that he is connected to VPN. 
It sits in the system tray and alerts you whenever a new non-ignored network connection is detected.

## Features

- Monitors active network connections continuously.
- Sends notifications via the tray icon when any new connection appears.
- Designed to detect VPN connection drops.
- Can run in the background and start automatically with Windows.
- Tray icon provides quick access to application controls.

## Usage

- Keep NetworkWatcher running in the background.
- Watch the tray icon for alerts about non-ignored connections.
